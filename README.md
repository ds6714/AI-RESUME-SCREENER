# 🧠 AI Resume Screener

An intelligent Resume Screening Web App built using **Python** and **Streamlit** that analyzes resumes and provides insights such as keyword matching, skill extraction, and candidate scoring.

---

## 🚀 Features

* 📄 Upload resumes in PDF format
* 🔍 Extract text from resumes
* 🧠 Analyze keywords based on job description
* 📊 Score resumes based on relevance
* 🏷️ Identify key skills and technologies
* ⚡ Simple and interactive UI using Streamlit

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Libraries Used:**

  * `streamlit`
  * `PyPDF2` / `pdfplumber`
  * `nltk` / `spacy`
  * `scikit-learn` (for similarity/scoring)

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/resume-screener.git
cd resume-screener
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
